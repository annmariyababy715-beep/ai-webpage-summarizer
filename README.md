AI Webpage Summarizer Chrome Extension

An AI-powered Chrome extension that extracts webpage content and generates concise summaries using the Groq API and LLM models.

Features

* Summarize webpages instantly
* AI-generated concise summaries
* Chrome Extension popup interface
* Fast response using Groq API

Tech Stack

* JavaScript
* HTML/CSS
* Chrome Extension APIs
* Groq API

Setup Instructions

1. Clone the repository
2. Open `popup.js`
3. Replace:

```javascript id="jlwm123"
const apiKey = "YOUR_GROQ_API_KEY";
```

with your Groq API key.

4. Open Chrome Extensions
5. Enable Developer Mode
6. Click "Load unpacked"
7. Select the project folder
